Column Headings for Linkage Information

Map data ID: Counter for data entries

Mutant: Name of embryo-defective mutant.

Marker: Name of visible marker

(#): Chromosome where visible marker is located

P: Percent recombination between EMB locus and visible marker

StdErr: Standard Error for percent recombination

Plants: Number of F2 plants screened for the marker and embryo phenotypes

Chi: Internal chi-square calculation.  This number gets smaller as the number of F2 plants observed in each phenotypic class
